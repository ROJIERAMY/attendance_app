استخدم Debug APK الجديد:
الملف: attendance-system-debug.apk (6.24 MB)

═══════════════════════════════════════════════════════
الحل الأسرع (3 خطوات):
═══════════════════════════════════════════════════════

1. انقل attendance-system-debug.apk إلى الجهاز اللوحي

2. على الجهاز:
   Settings → About tablet → اضغط 7 مرات على "Build number"
   Settings → Developer options → فعّل "USB debugging"
   Settings → Apps → Files → Install unknown apps (ON)

3. افتح Files → اضغط على APK → Install

═══════════════════════════════════════════════════════
إذا لم يعمل - استخدم ADB:
═══════════════════════════════════════════════════════

1. وصّل الجهاز بالكمبيوتر عبر USB

2. فعّل USB debugging على الجهاز

3. شغّل: install-via-adb.bat

═══════════════════════════════════════════════════════
رسائل الخطأ الشائعة:
═══════════════════════════════════════════════════════

"App not installed"
→ احذف أي إصدار سابق أولاً

"Package corrupt"
→ استخدم attendance-system-debug.apk

"Installation blocked"
→ فعّل "Install unknown apps" للتطبيق المستخدم
